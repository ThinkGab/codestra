## Milestone Requirements

### Hub File Routes

- [x] **FILE-01**: Hub espone `PUT /files/:swarmId/:filename` — salva file in memoria con UUID key, filename come metadata, risponde con `{id, filename, size, mimeType, uploadedAt}` — **Validated Phase 6**
- [x] **FILE-02**: Hub espone `GET /files/:swarmId/:filename` — supporta paginazione via `?offset=N&max_bytes=M`; risponde con `{content, offset, total_size, has_more}` — **Validated Phase 6**
- [x] **FILE-03**: Hub espone `GET /files/:swarmId` — lista tutti i file del swarm con `[{id, filename, size, mimeType, uploadedAt}]` — **Validated Phase 6**
- [x] **FILE-04**: Hub espone `DELETE /files/:swarmId/:filename` — rimuove file per filename nel namespace swarm; risponde con `{deleted: true}` — **Validated Phase 6**
- [x] **FILE-09**: Hub usa `crypto.randomUUID()` come Map key; filename client è solo metadata — nessun path traversal possibile — **Validated Phase 6**
- [x] **FILE-10**: Hub ha helper `readRawBody(req, limit)` separato da `readBody` per route upload (default 10 MB); risponde HTTP 413 su overflow — **Validated Phase 6**

### MCP Tools

- [x] **FILE-05**: `file_upload` tool in mcp-server.mjs — accetta `filename`, `content` (testo ≤50 KB), `mimeType` opzionale; usa swarm_id del worker come namespace; proxy a PUT hub — **Validated Phase 7**
- [x] **FILE-06**: `file_download` tool — accetta `filename`, `offset` (default 0), `max_bytes` (default 25000); restituisce content + has_more per paginazione — **Validated Phase 7**
- [x] **FILE-07**: `file_list` tool — restituisce lista file swarm corrente; proxy a GET /files/:swarmId — **Validated Phase 7**
- [x] **FILE-08**: `file_delete` tool — accetta `filename`; rimuove file nel swarm corrente; proxy a DELETE hub — **Validated Phase 7**

### Skills

- [x] **FILE-11**: `skills/codestra-file-transport/SKILL.md` creato con istruzioni per workers: quando usare file_upload/download, limiti size (testo ≤50 KB), semantica ephemeral (hub restart = file persi), pattern two-worker handoff via filename condiviso — **Validated Phase 8**
